## Change password

{{< numberingbox >}}

{{% numbering 1 %}} 
 Go to **Account -> Settings** and click **Profile**
{{% /numbering %}}

{{% numbering 2 %}} 
Click **Change password** link.
{{% /numbering %}}

{{% numbering 3 %}} 
Insert the new password into both *password* and *confirm password* field.
{{% /numbering %}}

{{% numbering 4 %}} 
Click **Update** button.
{{% /numbering %}}

{{< /numberingbox >}}
